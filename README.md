# E-ith

## Creative Admin Dashboard
A modern, responsive, and visually stunning React Admin Dashboard.

### Features
*   **Aesthetics**: Glassmorphism, smooth animations, customized scrollbars, and vibrant status-glow panels.
*   **Routing**: Dynamic view loading via React Router.
*   **Components**: Clean, reusable sidebar, navbar, and stat card components.
*   **Analytics**: Curated interactive Recharts (annual metrics line chart, spark lines, and weekly performance bar charts).
*   **Pages**:
    *   *Dashboard*: High-level operations overview.
    *   *Users*: Table supporting search and full state CRUD (Add/Edit/Delete).
    *   *Products*: Catalog with search/category pills and grid/list view toggles.
    *   *Orders*: Status logs showing items summary list and interactive state modifications.
    *   *Profile*: User detail sheet with flash toast save indicators.

### Tech Stack
*   React + JSX
*   Tailwind CSS v4 (with `@tailwindcss/vite` integration)
*   Recharts
*   React Icons
*   React Router v6
